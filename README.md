AgileLog — Sistema de Gerenciamento de Entregas Ágil

Empresa: TechFlow Solutions

O AgileLog é um sistema web simples e ágil desenvolvido como parte de um projeto acadêmico de Engenharia de Software, simulando o funcionamento de um ambiente logístico real baseado em metodologias ágeis.

Objetivo do Projeto

O AgileLog tem como objetivo facilitar o gerenciamento de entregas e tarefas logísticas dentro de uma startup de transporte, permitindo o acompanhamento de cada entrega em tempo real.

O sistema foi construído com base nos princípios de Scrum e Kanban, oferecendo uma visão visual e prática do fluxo de entregas.

Tecnologias Utilizadas

Python 3.13

Flask (framework web)

HTML, CSS e JSON

Git e GitHub (para versionamento e gestão do projeto)

GitHub Projects (para o quadro Kanban)

Mudança de Escopo do Projeto

Durante o desenvolvimento do AgileLog, o projeto passou por uma mudança de escopo planejada para representar um cenário real de negócios.

Inicialmente, o sistema foi idealizado como um gerenciador genérico de tarefas, mas evoluiu para um gerenciador de entregas logísticas, mantendo a filosofia ágil e aplicando práticas reais de gestão de software.

As principais alterações foram:

As tarefas se tornaram entregas;

Cada entrega possui descrição, motorista responsável e endereço de destino;

Os status foram adaptados ao processo logístico:

Aguardando Coleta

Em Rota

Entregue

O sistema simula o acompanhamento ágil de entregas, integrando planejamento, execução e monitoramento.
