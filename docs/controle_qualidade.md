# Controle de Qualidade - AgileLog

## Estratégia de Testes
- **Testes Unitários:** garantir que cada função (criar, editar, excluir tarefas) funcione individualmente.
- **Testes de Integração:** validar a comunicação entre as partes do sistema (interface e banco de dados).
- **Testes Automatizados:** executados via GitHub Actions para verificar a integridade do código a cada commit.

## Ferramentas Utilizadas
- **PyTest**: para escrever e executar os testes.
- **GitHub Actions**: para automação do pipeline.

## Benefícios
- Redução de erros humanos.
- Detecção rápida de falhas após atualizações.
- Garantia de qualidade antes da entrega final.
