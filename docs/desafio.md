# Desafio do Projeto - AgileLog

## Contexto
A TechFlow Solutions foi contratada por uma startup de logística para desenvolver um sistema de **gerenciamento de tarefas** baseado em metodologias ágeis.  
O cliente precisa acompanhar o fluxo de trabalho em tempo real, priorizar tarefas críticas e monitorar o desempenho da equipe de entregas.

## Objetivo
Criar um **sistema web simples de gerenciamento de tarefas**, que permita:
- Cadastrar tarefas
- Atualizar o status (A Fazer, Em Progresso, Concluído)
- Excluir tarefas concluídas
- Visualizar o andamento geral do time

## Escopo Inicial
O sistema terá:
1. Página de login simples (usuário e senha genéricos)
2. Tela principal com o **Kanban de tarefas**
3. CRUD básico (Criar, Ler, Atualizar e Excluir)

## Linguagem e Ferramentas
- **Linguagem:** Python  
- **Framework:** Flask (para simular o sistema web)  
- **Versionamento:** GitHub  
- **Metodologia:** Kanban (gestão visual no GitHub Projects)  
- **Testes automatizados:** PyTest  
- **Integração contínua:** GitHub Actions
