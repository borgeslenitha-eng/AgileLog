# Backlog do Produto - AgileLog

## Funcionalidades Principais
1. Criar tarefas com título e descrição  
2. Listar todas as tarefas cadastradas  
3. Editar informações de uma tarefa  
4. Excluir tarefas concluídas  
5. Filtrar tarefas por status

## Funcionalidades Futuras (Mudança de Escopo)
- Adicionar prioridade (Alta, Média, Baixa)
- Criar relatórios de produtividade
- Login com múltiplos usuários

## Critérios de Aceitação
- Todas as ações devem ser funcionais e refletir no banco de dados.
- O sistema deve ser acessível via navegador.
- Cada commit no GitHub deve descrever claramente a funcionalidade adicionada.
